# Music Store Admin 後台管理系統

這是一套樂器行的後台管理系統，具備會員管理、商品管理、訂單處理與營收統計等功能，採用 Vue 3 + Express 開發，支援 RWD 響應式設計。

## 使用技術

### 前端（Vue 3）
- Vue 3 + Composition API
- Vue Router
- Pinia（狀態管理）
- Element Plus（UI 元件庫）
- Tailwind CSS（RWD 響應式設計）
- Axios（API 串接）

### 後端（Node.js + Express）
- RESTful API 設計
- JWT 驗證機制
- Mongoose + MongoDB（資料儲存）
